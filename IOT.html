A COGNIFICATION APPROACH TO MEASURE CONTAMINATION IN LAKES

which used iot modules and ML for processing.

The project aims at providing a feasible and flexible solution to measure quality of the water. The main goal is to provide a complete view of the current state of contamination in water and create awareness regarding the importance of water safety.

Essential software and hardware elements used:

  • Arduino Uno Rev3
  • pH Sensor
  • Temperature Sensor
  • Turbidity Sensor
  • jupyter notebook
  • machine learning algorithm used :K-means Clustering, Decision Tree Classifier
